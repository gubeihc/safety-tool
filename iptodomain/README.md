# 旁站查询小工具

### 使用说明
环境最好是python3.9版本，遇到问题请提供截图和测试地址以供复现，接口失效问题是必然会遇到的这个后续摸鱼解决，需要添加接口可以提交issue。

#### 使用方式
  - python3 main.py -u 110.242.68.3
  - python3 main.py -f ip.txt
  - 默认保存到 result/ip.txt

### 现有接口
- ipdnsaizhan https://dns.aizhan.com

- ipipchaxun https://ipchaxun.com
